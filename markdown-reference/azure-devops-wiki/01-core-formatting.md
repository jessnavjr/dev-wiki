## Example

## Page title

> Quoted note

---

- Bullet

## Syntax

```md
# Page title

Line one with a forced break.  
Line two on the next rendered line.

> Quoted note

---

- Bullet
1. Ordered item
- [ ] Task item

<u>Underline in wiki HTML</u>
```

Notes:

- Azure DevOps uses two trailing spaces for a line break.
- Underline is available in wiki pages via HTML.
