# Azure DevOps Wiki Variation: HTML And Rich Text

## Example

<p>This text includes <del>removed</del> and <ins>inserted</ins> content.</p>
<font color="blue">Colored text in wiki pages</font>
<center>Centered text</center>
<p>This page uses <sup>superscript</sup> and <sub>subscript</sub>.</p>

<p>

Markdown can still work inside HTML blocks when spaced correctly.

</p>

## Syntax

```md
<p>This text includes <del>removed</del> and <ins>inserted</ins> content.</p>
<font color="blue">Colored text in wiki pages</font>
<center>Centered text</center>
<p>This page uses <sup>superscript</sup> and <sub>subscript</sub>.</p>

<p>

Markdown can still work inside HTML blocks when spaced correctly.

</p>
```

Notes:

- Azure DevOps wiki pages support several HTML tags for richer formatting.
- Leave blank lines where needed when mixing HTML blocks and Markdown.
